===========================================
PLACEHOLDERS USED:
===========================================
{{SEVERITY_COLOR}} - Color code (#dc2626 for critical, etc.)
{{SEVERITY_EMOJI}} - Emoji (🔴, 🟠, 🟡, 🔵)
{{SEVERITY}} - Severity level (CRITICAL, HIGH, etc.)
{{PRIORITY}} - Priority level (1 (Highest), etc.)
{{ALERT_TITLE}} - Alert title
{{STATUS}} - Alert status
{{ANOMALY_SCORE}} - Anomaly score
{{ALERT_ID}} - Alert ID
{{DESCRIPTION}} - Alert description
{{TIMESTAMP}} - Formatted timestamp
{{USER_EMAIL}} - User email
{{SOURCE_IP}} - Source IP address
{{ACTION}} - Action type
{{RESOURCE}} - Resource accessed
{{MODEL}} - Detection model name
{{DASHBOARD_URL}} - Link to dashboard
{{{unsubscribe}}} - SendGrid unsubscribe link
{{{unsubscribe_preferences}}} - SendGrid preferences link
